# 🎓 인공지능 응용 수업 발표 순서 추첨 사다리 게임

> 10명이 참가하는 웹 기반 사다리 게임으로, 인공지능 응용 수업의 발표 순서를 공정하게 결정합니다.

---

## 📁 파일 구성

```
sadari/
├── AI_사다리게임.html        ← 게임 본체 (이 파일 하나로 실행)
├── AI_사다리게임_대화기록.docx ← 제작 과정 대화 기록
└── README.md                 ← 사용 설명서 (현재 파일)
```

---

## 🚀 실행 방법

`AI_사다리게임.html` 파일을 **웹 브라우저(Chrome, Edge, Safari 등)로 열면** 바로 실행됩니다.
별도 설치나 인터넷 연결이 필요 없습니다.

---

## ⚙️ 게임 시작 전 설정

### 1. 발표자 이름 입력

화면 왼쪽 **"발표자 이름 설정"** 패널에서 참가자 10명의 이름을 입력합니다.

- 기본값: 김민준, 이서연, 박지호 … 등 10명
- 각 칸을 클릭하여 직접 수정 가능 (최대 6자)
- 입력 즉시 사다리 상단 태그에 반영됩니다

### 2. 발표 순서 슬롯 확인 (선택)

화면 오른쪽 **"발표 순서 슬롯 설정"** 패널에서 결과 항목을 변경할 수 있습니다.

- 기본값: 1번 발표 ~ 10번 발표
- 필요에 따라 자유롭게 수정 가능 (예: "1조 발표", "A팀" 등)

---

## 🎮 실행 방법 3가지

### ⚡ 전체 실행
모든 참가자의 경로를 **자동으로 순서대로** 애니메이션합니다.
버튼 하나로 10명 전원의 발표 순서가 한 번에 결정됩니다.

### ▶ 한 명씩 보기
버튼을 **누를 때마다 한 명씩** 순서대로 경로를 공개합니다.
긴장감 있게 한 명씩 발표 순서를 공개하고 싶을 때 사용하세요.

### 👆 개별 클릭
- 상단의 **이름 태그** 또는
- 중간의 **개별 버튼**을 클릭하면

원하는 참가자만 선택해서 경로를 확인할 수 있습니다.

---

## 🔄 초기화

**"🔄 초기화"** 버튼을 누르면 사다리 구조가 새로 생성되고 모든 결과가 초기화됩니다.
다시 추첨하고 싶을 때 사용하세요.

---

## 📊 최종 결과 확인

게임이 진행되면 화면 하단에 **"발표 순서 최종 결과"** 카드가 표시됩니다.

| 항목 | 내용 |
|------|------|
| 발표자 N | 참가자 번호 |
| 이름 | 참가자 이름 |
| N번 발표 | 배정된 발표 순서 |

경로가 공개될 때마다 카드가 하나씩 채워집니다.

## 🛠️ 기술 사양

- **파일 형식**: 단일 HTML 파일 (CSS·JS 내장)
- **외부 의존성**: 없음 (인터넷 불필요)
- **권장 브라우저**: Chrome, Edge, Safari, Firefox 최신 버전
- **화면 크기**: 데스크톱 권장 (모바일도 지원)

### 사다리 알고리즘
- **행 수**: 22행
- **가로선**: 행당 5~7개 (컬럼 균등 분배 알고리즘 적용)
- **경로 추적**: 수직·수평 웨이포인트 기반 정확한 경로 계산
- **공정성**: 매 초기화마다 랜덤 생성, 결과 예측 불가

## ❓ 자주 묻는 질문

**Q. 이름을 입력했는데 사다리에 반영이 안 돼요.**
A. 이름 입력란에 타이핑하면 실시간으로 반영됩니다. 반영이 안 될 경우 초기화 버튼을 눌러주세요.

**Q. 같은 결과가 두 번 나올 수 있나요?**
A. 아닙니다. 사다리 구조상 각 참가자는 반드시 서로 다른 결과에 도달합니다.

**Q. 이미 공개된 사람을 다시 클릭하면 어떻게 되나요?**
A. 이미 공개된 참가자는 재실행되지 않습니다.

**Q. 발표 순서를 다시 추첨하고 싶어요.**
A. "🔄 초기화" 버튼을 누르면 새로운 사다리로 다시 추첨할 수 있습니다.
